# Basic Usage Tutorial

Learn how to use md-mcp in just 10 minutes.

## Step 1: Installation

First, install the package:

```bash
pip install md-mcp
```

## Step 2: Configure

Point it at your markdown folder:

```bash
md-mcp --folder ~/docs --name "My Docs"
```

## Step 3: Use in Claude

Restart Claude Desktop and start asking questions about your docs!

That's it! You're ready to go.
